# JSChoilog

- **Git Commit 컨벤션 (Conventional Commit)**
  - Feat: 새로운 기능 추가
  - Fix: 버그 수정
  - Desing: CSS 등 사용자 UI 디자인 변경
  - Docs: 문서 수정
  - style: 코드 포맷팅 (비즈니스 로직 변경 없음)
  - Refactor: 리팩토링
  - Comment: 주석 추가 또는 변경
  - Chore: 빌드/패키지 관리 변경
  - Rename: 파일 또는 폴더 이름 수정/이동만 수행한 경우
  - Remove: 파일 또는 폴더 삭제만 수행한 경우
